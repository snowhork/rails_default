fig=docker-compose

### 最初にやる時

```
fig build
```

### サーバー立ち上げたい時

```
fig up
```

### コンテナの中に入りたい時

```
fig run [container name] bash
```

bundle install

```
fig run app bundle install
```

### railsコマンドを使いたい時

```
bin/spring [rails command]
```

### コンテナの中の任意のコマンドを実行したい時

```
fig run [container name] [command]
```

### binding.pryで対話実行したい時


(binding.pryで停止した後)
```
docker ps 
```
コンテナの名前を調べて（xxxxx_app_1など）
```
docker attach xxxxx_app_1
```
